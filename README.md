
# 2048 Game

A browser implementation of the classic 2048 puzzle game built with JavaScript, HTML, and SCSS.

## Demo

* [DEMO LINK](https://SoraEmpty.github.io/2048)

## Technologies Used

* HTML5
* SCSS
* JavaScript (ES6 Modules)
* Object-Oriented Programming (OOP)

## Features

* Classic 4×4 game board
* Tile movement using keyboard arrow keys
* Tile merging and score calculation
* Win and lose game states
* Restart game functionality
* Dynamic UI updates
* Responsive and clean interface

## How to Play

1. Press **Start** to begin the game.
2. Use the **Arrow Keys**:

   * ⬆️ Up
   * ⬇️ Down
   * ⬅️ Left
   * ➡️ Right
3. Tiles with the same value merge when they collide.
4. Reach the **2048** tile to win.
5. The game ends when no more valid moves are available.

## Project Structure

* `index.html` - application markup
* `scripts/` - game initialization and UI logic
* `modules/` - game mechanics and classes
* `styles/` - SCSS styles

## Getting Started

1. Clone the repository:

   git clone https://github.com/SoraEmpty/2048.git

2. Install dependencies:

   npm install


3. Run the project locally:

   npm start
